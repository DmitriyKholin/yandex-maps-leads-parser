# 🗺️ Yandex Maps Leads Parser (B2B Lead Generation)

Асинхронный парсер Яндекс.Карт для автоматического сбора B2B-лидов и генерации баз для холодных продаж (аутрича).
Инструмент обходит блокировки, собирает контактные данные организаций и автоматически генерирует прямые ссылки для связи в Telegram.

## 🚀 Фичи (Особенности)
- **Асинхронность:** Работает на базе `Playwright` и `asyncio`, что обеспечивает быструю загрузку страниц.
- **Обход блокировок:** Уникальная логика сбора ID организаций и изолированные вкладки браузера предотвращают краши (Out of Memory) и капчи.
- **Умная фильтрация телефонов:** Автоматически определяет мобильные номера (РФ) и генерирует кликабельную ссылку вида `https://t.me/+7999...` для быстрой отправки сообщений в Telegram Desktop.
- **Валидация данных:** Строгая типизация "под капотом" с помощью `Pydantic`.
- **Экспорт в Excel:** Удобное сохранение результатов в `.xlsx` формат через `pandas`.

## 🛠 Технологический стек
- Python 3.10+
- Playwright (Chromium)
- Pydantic v2
- Pandas & Openpyxl
- Loguru (для логирования)

## ⚙️ Установка и запуск
```bash
1. Клонируйте репозиторий:

git clone https://github.com/ВАШ_НИК/leads_parser.git
cd leads_parser
2. Создайте и активируйте виртуальное окружение:
python -m venv venv

# Для Windows:
venv\Scripts\activate

# Для Mac/Linux:
source venv/bin/activate
3. Установите зависимости:
pip install -r requirements.txt
4. Установите браузер для Playwright:
playwright install chromium
5. Запустите парсер:
python main.py
При запуске скрипт попросит ввести нишу (например, "салоны красоты") и город.
```
